This app is developed using
	Springboot,Spring JPA,Angular8

To start springboot app.
	Import backend folder in eclipse
	Start as springboot application
	App will be running in 8080 port by default
	To check access : http://localhost:8080/api/swagger-ui.html			

To start Angular app locally
	Open Frontend folder in Vscode(or any editor of your choice)
	npm install
	to start applicaion run : npm start (proxy is configured to consume api from local)
